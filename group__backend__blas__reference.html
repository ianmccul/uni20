Detailed Description

Header-only wrappers that provide a portable BLAS fallback implementation.

The reference backend expands the generic BLAS shims defined in detail/blasproto.hpp for each supported scalar type. These wrappers are primarily used for tests and environments where no optimized vendor library is available.
