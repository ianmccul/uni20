Detailed Description

template<typename T>
class uni20::async::EpochContextWriter< T >

RAII-scoped representation of a writer’s participation in an epoch.

Constructed by EpochQueue and passed into a WriteBuffer<T>, this class manages the binding and release of a writer coroutine to a single EpochContext. It ensures that the write gate is completed exactly once, either manually via release() or automatically on destruction.
