◆ help_while_waiting()

virtual void uni20::async::IScheduler::help_while_waiting ( const WaitPredicate &  is_ready)
inlinevirtual

Allow a scheduler to advance queued work while a thread is blocking.

Blocking waits (e.g., Async<T>::get_wait()) call this hook to cooperatively drive progress on the owning scheduler until is_ready() reports completion. The default implementation simply yields the calling thread until the predicate succeeds, which is suitable for schedulers that rely on dedicated worker threads.

Reimplemented in uni20::async::DebugScheduler, and uni20::async::TbbScheduler.

virtual void uni20::async::IScheduler::wait_for ( const WaitPredicate &  is_ready)
inlinevirtual

Block the calling thread until is_ready returns true.

Implementations may override this to provide scheduler-specific waiting semantics (e.g., parking on a condition variable). The default implementation repeatedly invokes help_while_waiting until the predicate succeeds.
