Detailed Description

template<typename T, std::size_t MR, std::size_t NR, std::size_t KR>
class uni20::kernel::cpu::GemmLoop< T, MR, NR, KR >

Generic M×N×K contraction engine with full K-dimension recursion.

Each dimension group stores extents and operand strides used to perform fused tensor contractions without materialising temporaries.
