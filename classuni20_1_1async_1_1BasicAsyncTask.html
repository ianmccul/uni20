Abandon/leak the coroutine.

Releases ownership and marks the task as intentionally leaked.

This is useful only for emergencies, eg to unwind the stack without causing further errors.

Precondition

template<IsAsyncTaskPromise Promise>
BasicAsyncTask< T > uni20::async::BasicAsyncTask< T >::make_sole_owner ( BasicAsyncTask< Promise > &&  task)
static

Retain the task only if it is the sole remaining owner.

Converts a possibly-shared task into a sole-owner task handle.

This operation checks the awaiter count. If this is the last owner, then return the task unchanged. Otherwise, releases the task and leaves it equivalent to a moved-from state.

Returns
The original task if it had exclusive ownership; otherwise a null task.
Template Parameters
TPromise type.
Parameters
taskTask whose ownership should be normalized.
Returns
Sole-owner task if ownership transfer succeeded, otherwise an empty task.

◆ reschedule()

template<IsAsyncTaskPromise Promise>
void uni20::async::BasicAsyncTask< T >::reschedule ( BasicAsyncTask< Promise >  task)
static

Resubmit a suspended BasicAsyncTask to its scheduler, if this is the sole remaining owner.

Reschedules a task when this instance becomes its sole owner.

This transfers ownership to the scheduler only if the task has exclusive ownership of the coroutine. If other awaiters remain, the task is discarded and not rescheduled.

Precondition
The scheduler in the promise must have been set.
Parameters
taskThe task to be conditionally rescheduled.
