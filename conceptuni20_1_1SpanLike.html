Detailed Description

A “span-like” type usable by uni20’s zip_transform machinery.

A type S models SpanLike if it provides the minimal mdspan-like API:

  • S::extents_type defines the shape type
  • S::layout_type defines the layout policy
  • S::accessor_type defines the accessor policy
  • a const mapping() member returning something convertible to layout_type::mapping<extents_type>
  • a const data_handle() member returning something convertible to accessor_type::data_handle_type
  • an accessor() member returning something convertible to accessor_type
