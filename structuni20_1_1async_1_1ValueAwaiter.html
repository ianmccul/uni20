Detailed Description

template<typename T>
struct uni20::async::ValueAwaiter< T >

Awaitable wrapper for scalar values used in async expressions.

Allows scalar values to participate in co_await expressions used by async kernels. This awaitable never suspends, and simply returns the value when resumed. We store a value here, which means that we necessarily copy (or move). This is fine, since we have have an expression involving an Async<T> x, and a concrete value y, we must copy (or move) y into the coroutine / scheduler anyway.
