Detailed Description

template<typename T>
class uni20::async::EpochContextReader< T >

RAII handle representing the writer of a value in an EpochContext.

On construction, this marks the epoch as having a writer. When the buffer is set and the writer releases the value, waiting readers may resume.

EpochContextWriter does not represent a coroutine or suspendable operation. It is a scoped object used to enforce single-writer discipline and ensure writer→readers sequencing.

Note
Writer must eventually call .release() to trigger reader advancement.
