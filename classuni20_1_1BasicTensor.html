Detailed Description

template<typename ElementType, typename Extents, typename StoragePolicy = VectorStorage, typename LayoutPolicy = stdex::layout_stride, typename AccessorFactory = DefaultAccessorFactory>
class uni20::BasicTensor< ElementType, Extents, StoragePolicy, LayoutPolicy, AccessorFactory >

Owning tensor that allocates storage and exposes mdspan-based access.

Template Parameters
ElementTypeValue type stored by the tensor.
ExtentsExtents type describing the tensor shape.
StoragePolicyPolicy controlling ownership and allocation of the buffer.
LayoutPolicyLayout policy that determines index ordering and stride computation.
AccessorFactoryFactory that produces accessors for the storage handle.
