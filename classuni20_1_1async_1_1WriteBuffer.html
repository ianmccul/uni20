Detailed Description

template<typename T>
class uni20::async::WriteBuffer< T >

Awaitable write buffer for one Async<T> epoch.

co_await writer yields a borrowed proxy tied to the WriteBuffer<T> object. co_await writer.transfer() yields an owning proxy whose lifetime no longer depends on the original buffer object.
