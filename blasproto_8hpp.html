Detailed Description

Templated BLAS interface declarations and wrappers.

This header defines overloads for BLAS level 2 and 3 routines such as gemm, gemv, syrk, herk, etc., using Fortran-style symbols like dgemm_, zher2k_, and so on.

It is designed to be included multiple times, with the following preprocessor macros defined before each inclusion:

  • BLASCHAR: One of s, d, c, or z, indicating the BLAS prefix.
  • BLASTYPE: The C++ scalar type corresponding to the BLAS prefix.
  • BLASREALTYPE: The corresponding real type, used by herk/her2k.
  • BLASCOMPLEX: Either 0 (for real types) or 1 (for complex types).

Each inclusion generates extern "C" declarations and inline C++ overloads, optionally including trace hooks via UNI20_API_CALL.

Note
This file must not use #pragma once or include guards.
Valid combinations of macros:

DOUBLE PRECISION d float64 float64 0
