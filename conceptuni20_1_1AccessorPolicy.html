Detailed Description

A model of the C++ mdspan AccessorPolicy named requirement.

Requirements for AP:

  • nested types: element_type, data_handle_type, offset_policy, reference.
  • offset(dh, off) must be constexpr noexcept and return a type convertible to offset_policy::data_handle_type.
  • access(dh, off) must be constexpr noexcept and return exactly AP::reference.
