Detailed Description

template<typename T, typename Traits>
class uni20::TensorView< T, Traits >

Trait bundle describing the policies required to build tensor views.

Tensor view specialisation providing mutable access.

Template Parameters
ExtentsExtents type describing the tensor shape.
StoragePolicyPolicy that identifies how storage is owned or referenced.
LayoutPolicyLayout policy that maps indices to offsets.
AccessorPolicyAccessor policy used to interact with the storage handle.

Forward declaration for the TensorView template using bundled traits.

Template Parameters
ElementTypeValue type viewed by the tensor.
TraitsTrait bundle describing policies and extents.
TValue type stored in the tensor.
TraitsTrait bundle describing policies and extents.
